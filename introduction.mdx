# Welcome to MiTienda Web

MiTienda Web is a full-featured e-commerce web application built with ASP.NET Core 9 MVC and SQL Server. It provides a complete solution for managing online stores with product catalogs, shopping carts, user accounts, and order processing.

## Technology Stack

MiTienda Web is built with modern, proven technologies:

- **ASP.NET Core 9** - High-performance web framework with MVC pattern
- **Entity Framework Core** - Modern ORM for database operations
- **SQL Server** - Enterprise-grade relational database
- **Cookie Authentication** - Secure session management
- **Bootstrap** - Responsive UI framework

## Use Cases

MiTienda Web is perfect for:

- **Small to Medium Businesses** - Launch your online store quickly
- **Learning Projects** - Understand e-commerce architecture patterns
- **Prototyping** - Build MVPs for e-commerce concepts

## Project Structure

```
MiTienda_practica/
├── Controllers/          # MVC controllers handling HTTP requests
├── Models/              # View models for data transfer
├── Views/               # Razor views for UI
├── Entities/            # Database entities
├── Services/            # Business logic layer
├── Repositories/        # Data access layer
├── Context/             # EF Core database context
├── Migrations/          # Database migrations
├── Utilities/           # Helper classes and extensions
└── wwwroot/            # Static files (CSS, JS, images)
```
